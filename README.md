## 贡献审核
审核要求如下，如果不符合会被打回！

1. 不能有语法错误

2. 内容健康、向上

3. 内容可以是官方想做的，或者是有创意的

4. 不可以对官网内容做大幅度修改
## 贡献代码修改规则
请按照以下方式修改代码，不符合的将被打回！

文本：

```html
<p>文本</p>
<h2>文本</h2>
```

图片等链接和css样式：

```html
<img src="./img/图片名字.图片格式">
```
```css
body {
  background-size: 100%;
}
```

当需要使用**其他创意性内容**，可使用`<div>`等。
